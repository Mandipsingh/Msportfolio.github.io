About

Rajastan select is a feature introduced in the 2018 Pantone collection. This collection is inspired from a city in India, which is known for its vibrant colors.This piece of work takes the viewer through a set of Informative and interactive photographs and abstract pieces that define the ratios and provide Pantone color codes for complementary and contrasting colors. Designers and Artists are welcomed to use these picked Pantones for a theme that evokes this vibrant culture.
